# Prompts Index

This folder contains ordered prompts for building and refining the Learn AI the American Way Astro tutorial site.

Use prompts in numeric order unless you are intentionally rerunning a specific task.

## How to use this folder

1. Open your Astro site repository in VS Code.
2. Open one prompt file from this folder.
3. Copy the full prompt text.
4. Paste it into Copilot Chat.
5. Review generated changes in the site repo.
6. Run local build checks.
7. Commit, then continue to the next prompt.

## Phase guide

2. Base file generation.
3. Astro starter scaffolding.
4. Early review and validation.
5. Initial core content.
6. Internal clarity and links.
7. Life stage page expansion.
8. Tool guide content.
9. Beginner readability audit.
10. GitHub Pages deployment prep.

### 11 to 25 Core site expansion

1. Homepage design refinements.
2. Lesson and prompt library growth.
3. SEO passes.
4. Link and trust audits.
5. About, mission, FAQ, and use case hubs.

### 26 to 50 Advanced content and launch readiness

1. Learning path and safety prompting guide.
2. Navigation and homepage messaging refinement.
3. Specialized audience pages and content clusters.
4. Deployment docs, maintenance workflow, and launch checklist.

### 51 to 75 Everyday use packs and structural audits

1. New practical use pages.
2. SEO cluster refinements.
3. Readability and scannability audits.
4. Page template and section design refinements.
5. Publishing process and hierarchy improvements.
