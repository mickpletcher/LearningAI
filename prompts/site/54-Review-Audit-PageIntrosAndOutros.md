# Prompt 54

You are working inside an Astro website project called Learn AI the American Way.

Your task:
Audit and improve page intros and page endings across the site so each page starts clearly and ends with a useful next step.

Goal:
Make every page easier for beginners to enter, understand, and continue from.

Files to review:

- all files in src/pages/

- all Markdown files in src/content/lessons/

- shared components that include repeated intro or outro style copy if any exist

Audit focus:

1. whether intros quickly explain what the page is for

2. whether intros speak in plain English

3. whether intros are too vague, too long, or too generic

4. whether endings give a clear next step

5. whether endings feel natural and helpful

6. whether pages miss chances to guide users to the next useful topic

Tasks:

- rewrite weak intros

- tighten bloated intros

- improve vague openings

- improve page endings so the next step feels clear

- add or improve links to related content at the end of pages

- make intros and endings more consistent in quality across the site

- reduce repetitive intro and outro phrasing where it feels copied

Tone requirements:

- plainspoken

- practical

- calm

- direct

- beginner friendly

- respectful

- not robotic

- not corporate

- no hype

Important constraints:

- do not make every intro sound identical

- do not make every ending a generic read more

- do not remove useful context just to shorten intros

- do not generate a report without making the fixes directly

Implementation requirements:

- directly update the relevant files

- preserve Astro structure and working links

- keep the user journey stronger from page to page

Output requirement:
Directly update the relevant files to improve page intros and page endings.
Do not summarize.
Do not explain.
Fix the intros and outros.
