# Prompt 75

You are working inside an Astro website project called Learn AI the American Way.

Your task:
Refine the site's section headings and spacing so pages feel easier to scan, visually calmer, and more polished.

Goal:
Make the site more readable and structured without redesigning it or making it look flashy.

Files to review and update:

- all files in src/pages/

- shared components that control section headings or page sections

- src/styles/global.css or the current global stylesheet location

- src/layouts/BaseLayout.astro if needed

Design and readability goals:

1. improve heading clarity

2. improve visual spacing between sections

3. improve consistency of heading styles

4. improve the page rhythm so content breathes better

5. make long pages easier to navigate visually

6. keep the design calm and practical

Tasks:

- improve vague or generic section headings

- standardize spacing patterns where appropriate

- tighten crowded sections

- reduce unnecessary empty space where it feels loose

- improve heading hierarchy visually

- make section transitions feel cleaner

- improve prompt box and callout spacing if needed

Style guidance:

- clean

- readable

- consistent

- calm

- practical

- beginner friendly

- not flashy

- not overdesigned

Important constraints:

- do not change content meaning just to alter headings

- do not make every heading sound the same

- do not redesign the whole theme

- do not add decorative clutter

- do not overengineer the CSS

Implementation requirements:

- directly update the relevant files and styles

- preserve Astro structure and GitHub Pages compatibility

- keep the design system maintainable

- improve spacing and headings only where it clearly helps readability

Output requirement:
Generate the actual updated file contents for the relevant pages, components, and styles where needed.
Do not summarize.
Do not explain.
Refine the section headings and spacing.
