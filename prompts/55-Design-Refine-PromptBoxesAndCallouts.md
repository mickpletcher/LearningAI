# Prompt 55

You are working inside an Astro website project called Learn AI the American Way.

Your task:
Refine the prompt boxes and supporting callout styles so they are easier to spot, easier to read, and more useful for beginners.

Goal:
Make prompt examples and important callouts feel practical, readable, and visually consistent across the site.

Files to review and update:

- src/components/PromptBox.astro

- any pages using prompt boxes

- any shared components or styles used for watch out sections, tip boxes, or related callouts

- src/styles/global.css or the current global stylesheet location

Prompt box and callout goals:

1. make prompt examples clearly recognizable

2. improve readability of copied prompts

3. improve spacing and typography

4. improve visual distinction between prompts, tips, and warnings

5. keep the site calm and uncluttered

6. improve mobile readability of long prompts

Tasks:

- refine the PromptBox component if needed

- improve labels such as Try this prompt, Watch out, Tip, or Next step

- improve spacing and layout for prompt examples

- improve styling for warning or watch out callouts so they are visible but not harsh

- improve copy around prompt examples where needed so users know what to do with them

- standardize repeated callout patterns across the site

Design guidance:

- clean and simple

- easy to scan

- strong readability

- calm visual hierarchy

- subtle emphasis rather than flashy design

- accessible contrast and spacing

- mobile friendly formatting

Tone requirements:

- practical

- direct

- beginner friendly

- calm

- not robotic

- not corporate

- no hype

Important constraints:

- do not add heavy JavaScript just for copying buttons unless already present and truly useful

- do not make callouts visually loud or distracting

- do not clutter pages with too many box styles

- do not weaken important warnings

- do not redesign the whole site

Implementation requirements:

- directly update the relevant files and styles

- preserve Astro structure and reusable component patterns

- keep the system maintainable and consistent

- ensure the design works well on desktop and mobile

Output requirement:
Generate the actual updated file contents for the relevant components, pages, and styles where needed.
Do not summarize.
Do not explain.
Refine the prompt boxes and callouts.
