# LearningAI Lesson Library

This folder contains 40 interactive markdown-based lessons for learning AI fundamentals and practical skills.

Each lesson is a self-contained markdown file designed to be used as a guided learning session in ChatGPT, Claude, Gemini, or Copilot. No coding required. No installation needed. Copy the lesson file, paste it into your AI tool of choice, and start learning.

## How to use a lesson

1. Open a lesson markdown file from this folder
2. Copy all the text from the file (Ctrl+A, Ctrl+C)
3. Go to ChatGPT, Claude, Gemini, or Copilot
4. Paste the lesson text (Ctrl+V)
5. Press Enter and follow along

The AI will guide you through the lesson step by step. Some lessons support commands like "continue", "recap", "restart", or "ask me a question" — the lesson file will explain what commands are available.

Each lesson typically takes 15-30 minutes. You control the pace.

## Lesson format

Each lesson markdown file includes:

- **Assistant instructions** — tells the AI how to teach this lesson
- **Metadata** — topic, audience level, and learning outcomes
- **Lesson body** — 2-5 content sections (typically 200-400 words each)
- **Try this yourself** — 2-3 practical exercises
- **What to watch out for** — cautions and real problems to avoid
- **Check your understanding** — 3 self-check questions with answers
- **What to learn next** — links to related lessons

This consistent structure makes lessons portable across different AI tools while maintaining quality and pacing. You get the same learning experience whether you use ChatGPT, Claude, Gemini, or Copilot.

## Getting stuck

If a lesson feels confusing:

1. Ask the AI to explain it again in simpler language
2. Ask for one real-world example
3. Skip to the next lesson and come back later
4. Copy and paste the lesson again and start over

There is no penalty for moving slowly or revisiting lessons.

## Creating new lessons

To write a new lesson for this library:

1. Open [lesson-template.md](lesson-template.md)
2. Follow the instructions and fill in all sections with substantive content
3. Save as `NN-topic-name.md` in this folder (use the next available number)
4. Update this README to include your lesson in the catalog
5. Add cross-references in the "What to learn next" section of related lessons

All lessons must follow the writing requirements and tone guidelines in [lesson-template.md](lesson-template.md).
