# LearningAI

LearningAI is a simple prompt library.

It helps people learn AI in plain English.

## Start here if you are new

If you only want to learn AI, follow these steps.

1. Open [prompts/lessons/README.md](prompts/lessons/README.md).
2. Pick one lesson.
3. Open that lesson file.
4. Copy all text in the file.
5. Paste it into ChatGPT, Claude, Gemini, or Copilot.
6. Read the lesson output.
7. Repeat with the next lesson.

No coding setup is needed for lesson use.

## How to use this repo

Use this if you want to learn AI quickly.

1. Start with [prompts/lessons/README.md](prompts/lessons/README.md).
2. Work through lessons in number order.
3. Save your favorite prompts in a notes app.
4. Reuse prompts with your own real tasks.

## Common questions

### Do I need to install anything to use lessons

No.

You only need an AI chat tool and this repo.

### Which lesson should I start with

Start with [prompts/lessons/01-what-is-ai.md](prompts/lessons/01-what-is-ai.md).

Then continue in order.

### I feel stuck

Go back one lesson.

Copy the prompt again.

Ask the AI to explain in simpler words and use one real life example.
