# Lesson: How to Fact Check AI Answers

## Introduction and context

As AI tools become common, the real advantage is not using AI more often. It is using AI more wisely.

This means checking quality, understanding limits, and applying judgment before taking action.

Strong workflows reduce errors, protect sensitive information, and improve confidence.

## Why AI is useful for this topic

AI is useful here because it helps with verifying claims before relying on AI output.

It can quickly structure ideas and surface options, but your review is what makes the result reliable.

Example prompt:

Help me with How to Fact Check AI Answers. Keep it practical, explain risks, and provide a step by step approach.

## How to ask for better help on this topic

Weak prompt:
Is this correct?

Better prompt:
Help me fact check this AI answer by identifying claims, suggesting trusted sources, and listing what needs verification.

A stronger prompt includes context, constraints, and what success should look like.

## Practical examples

1. Extract claims that need verification.

Example prompt:
List factual claims in this response and mark which ones need external verification first.

1. Create a source checking plan.

Example prompt:
Build a source checking plan for these claims using reliable source types.

1. Cross check conflicting information.

Example prompt:
Compare these two sources and explain which appears more reliable and why.

1. Write a verified summary.

Example prompt:
Rewrite this answer using only points that can be verified from reliable sources.

## Try this yourself

1. Pick one real task and apply this lesson method to it.
2. Improve your prompt with context, constraints, and quality checks.
3. Review and validate the output before using it in real decisions.

## What to watch out for

1. Trusting confident answers without verification.
2. Sharing sensitive details that should stay private.
3. Skipping review because the output sounds polished.
