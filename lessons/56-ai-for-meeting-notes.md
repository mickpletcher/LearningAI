# Lesson: AI for Meeting Notes

## Introduction and context

This topic matters because teams lose time when information is unclear, scattered, or hard to act on.

AI helps you move from rough input to useful output faster.

You still decide what is correct, useful, and ready to share.

## Why AI is useful for this topic

AI is useful here because it helps with capturing key decisions and action items quickly.

It can turn rough notes into structured drafts, suggest better wording, and adapt tone for different audiences.

Example prompt:

I need help with AI for Meeting Notes. Use plain language and give me a practical output I can use at work today.

## How to ask for better help on this topic

Weak prompt:
Take notes from this meeting.

Better prompt:
Turn these rough meeting notes into a clean summary with decisions, owners, deadlines, and open questions.

A better prompt usually includes audience, goal, format, and constraints.

## Practical examples

1. Convert rough notes into a clear summary.

Example prompt:
Clean up these notes into sections for decisions, tasks, risks, and follow ups.

2. Create an action item tracker from the notes.

Example prompt:
Extract action items with owner, due date, and status from these notes.

3. Generate a short recap email for attendees.

Example prompt:
Write a recap email from these notes using a clear and professional tone.

4. Produce different versions for manager and team.

Example prompt:
Create two versions of this recap: one executive summary and one detailed team update.

## Try this yourself

1. Use a real task from this week and ask AI for a first draft.
2. Ask AI to improve clarity and structure for your audience.
3. Review the output and make final edits in your own voice before sharing.

## What to watch out for

1. Sharing AI output without checking facts, names, dates, and ownership.
2. Using vague prompts that create generic answers.
3. Forgetting to add context about audience, timeline, and constraints.

## Check your understanding

### Question 1
What makes a prompt better for workplace tasks like this?

Answer: Include audience, purpose, output format, and constraints so the result is relevant and usable.

### Question 2
Why should you review AI output before using it?

Answer: AI can miss context or include mistakes, so human review protects quality and accuracy.

### Question 3
What is a simple workflow for this lesson topic?

Answer: Provide context, generate a draft, refine for clarity, then finalize with your own judgment.
